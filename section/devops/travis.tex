\section{Travis}

\index{Travis}

Validating a travis file:

Online service: \url{http://lint.travis-ci.org/}

Commandline Install

\begin{lstlisting}
gem install travis --no-rdoc --no-ri
\end{lstlisting}

Commandline Use

\begin{lstlisting}
travis lint <path to your .travis.yml>
\end{lstlisting}
